# Song-To-Drumsheet

Eine Android-App, die aus einem Lied automatisch Schlagzeugnoten generiert.

## Funktionen

- **Audio-Eingabe**: Laden Sie MP3, WAV oder andere Audiodateien
- **Beat-Erkennung**: Automatische Analyse des Rhythmus und der Schlagzeugmuster
- **Noten-Generierung**: Erstellt lesbare Schlagzeugnoten aus dem analysierten Audio
- **Export**: Speichern und teilen Sie die generierten Noten

## Features

- **Audio Input**: Load MP3, WAV or other audio files
- **Beat Detection**: Automatic rhythm and drum pattern analysis
- **Note Generation**: Creates readable drum sheet music from analyzed audio
- **Export**: Save and share the generated notes

## Technologie-Stack

- **Android**: Native Android App (Kotlin)
- **Audio-Verarbeitung**: TarsosDSP für Audio-Analyse
- **Beat-Erkennung**: BPM-Detektion und Onset-Erkennung
- **UI**: Material Design 3

## Installation

1. Klonen Sie das Repository
2. Öffnen Sie das Projekt in Android Studio
3. Bauen und installieren Sie die App auf Ihrem Android-Gerät

## Verwendung

1. Starten Sie die App
2. Wählen Sie eine Audiodatei von Ihrem Gerät
3. Die App analysiert automatisch das Lied
4. Schlagzeugnoten werden generiert und angezeigt
5. Exportieren Sie die Noten als PDF oder Bild
